# Numerical Project 1

nmr.f95 reads 'nmr.in' by default

Format for read in file:

File input
--------------
Name of NMR file
Baseline adjustment
Tolerance for algorithms
Filter type (0=None, 1=Boxcar, 2=SG)
Size of Filter
Filter passes
Integration (0=Newton-Cotes,1=Romberg,2=Adaptive,3=Quadrature) 
output file
